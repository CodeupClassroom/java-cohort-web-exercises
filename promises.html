** REQUEST MADE **

pending

Pending

  • Request was made
  • request is pending any kind of response
  • don't quite know yet what that response will be

check_circle

Success

  • request has been made successfully
  • no errors in the request
  • no errors in the response
  • request is made successfully
  • client receives expected outcome

error

Fail

  • request has been made
  • either request or response has been unsuccessful
  • error is returned to the user (100,200,300,400,500)
  • error can have additional info about the error itself
